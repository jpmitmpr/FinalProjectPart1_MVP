# Final Project API

## Description
A RESTful API built with Express and Sequelize featuring full CRUD operations, JWT-based authentication, role-based authorization, ownership checks, pagination, and filtering.

---

## Features
- User & Post CRUD operations
- JWT Authentication (Register / Login / Logout)
- Password hashing with bcrypt
- Role-based access control (user / admin)
- Ownership enforcement
- Pagination & filtering
- Comprehensive unit tests
- Deployed on Render

---

## Tech Stack
- Node.js
- Express
- Sequelize
- SQLite
- JWT
- bcrypt
- Jest / Supertest
